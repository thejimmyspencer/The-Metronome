# Pulse Setlist Metronome

This folder is ready for GitHub Pages.

## Publish with GitHub Pages

1. In GitHub, create a **public** repository named `metronome`.
2. Upload every file and folder inside this folder to the repository's top level. Do not upload the folder itself as an extra nested layer.
3. In the repository, select **Settings**, then **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch **main** and folder **/(root)**, then choose **Save**.
6. Wait for GitHub to show the published address, then open it on your phone.

On iPhone, tap Share and choose **Add to Home Screen**. On Android, use the browser menu and choose **Install app** or **Add to Home Screen**.

Set lists and settings are saved on each phone using its browser storage. They are not shared between devices.
